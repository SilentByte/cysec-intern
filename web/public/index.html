Controls: WASD to move, E to interact, mouse to select options.
CySec Intern is a casual game where you explore the world as a newly hired intern and learn more about cybersecurity concepts on your journey. It has been created for the Secureworks® Cybersecurity Literacy Challenge.
This game has been created using the Godot Game Engine and the source code is available on Github @SilentByte.
